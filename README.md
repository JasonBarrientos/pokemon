## Description

App de pokemon basado en pokeapi

## Project setup

```bash
 npm install
```

## Run base

```bash
# instalar cli
 npm i -g @nestjs/cli

# levantar base de datos
 docker-compose up -d

```
## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

```
## Reconstruir la base ded atos con la semilla
```
http://localhost:3000/api/seed
```

## Stack Usado
* MongoDB
* Nestjs
